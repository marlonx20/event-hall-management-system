# Event Hall Management System

A web-based management system designed to help event hall owners manage reservations, customers, payments, reminders and daily operations from a single application.

This project was created as a real-world software engineering project, following a structured development process including requirements analysis, system design, implementation, testing and documentation.

---

## Project Status

🚧 Under Development

Current phase:

- Business Analysis

---

## Motivation

The current reservation process relies on multiple disconnected tools:

- Facebook
- Facebook Messenger
- WhatsApp
- Phone calls
- Microsoft Excel

Managing reservations across different platforms increases manual work and the possibility of human errors.

The goal of this project is to centralize the entire workflow into a single web application.
