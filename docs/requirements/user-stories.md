# 1. Introduction

This document describes the user stories for the Event Hall Management System.

Each story represents a feature from the perspective of the primary user.

---

## US-001 - Check Availability

**As** the event hall owner,

**I want** to quickly check whether a date is available,

**So that** I can answer customer inquiries immediately.

---

## US-002 - Register Reservation

**As** the event hall owner,

**I want** to register a new reservation,

**So that** I can stop using Excel.

---

## US-003 - View Calendar

**As** the event hall owner,

**I want** to see all reservations in a calendar,

**So that** I can quickly identify occupied and available dates.

---

## US-004 - Register Payments

**As** the event hall owner,

**I want** to register deposits and remaining payments,

**So that** I always know how much each customer still owes.

---

## US-005 - Store Customer Requirements

**As** the event hall owner,

**I want** to record special customer requirements,

**So that** nothing is forgotten before the event.

---

## US-006 - Create Internal Reminders

**As** the event hall owner,

**I want** to create reminders,

**So that** maintenance tasks are not forgotten.

---

## US-007 - Copy Frequently Used Information

**As** the event hall owner,

**I want** to copy predefined messages and business information,

**So that** I can respond to customers more quickly.

---

## US-008 - View Pending Payments

**As** the event hall owner,

**I want** to immediately see which customers still have pending balances,

**So that** I can collect payments before each event.

---

## US-009 - Prepare Upcoming Events

**As** the event hall owner,

**I want** to see today's and upcoming events,

**So that** I can prepare everything needed in advance.

---

## US-010 - Manage the Business from Anywhere

**As** the event hall owner,

**I want** to access the application from my phone or computer,

**So that** I can manage reservations wherever I am.
