# Dashboard

## Purpose

The dashboard is the main entry point of the application.

It should provide the owner with all important information needed to manage the business on a daily basis.

---

# Sections

## Today's Events

Displays today's reservations.

Each reservation should show:

- Customer
- Event type
- Start time
- Remaining balance
- Special requirements

---

## Pending Payments

Displays reservations with outstanding balances.

---

## Upcoming Reservations

Displays reservations for the next few days.

---

## General Reminders

Displays maintenance reminders.

Examples:

- Repair restroom
- Buy chairs
- Wash tablecloths

---

## Quick Actions

- Create Reservation
- Open Calendar
- View Customers
- Add Reminder
